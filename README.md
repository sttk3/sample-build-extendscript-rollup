# sample-build-extendscript-rollup

RollupとBabelを使い、モジュールごとに別ファイルに切り分けたExtendScriptを1つのjsxファイルとして生成できるようにするサンプルプロジェクトです。

ついでにES5以降の記法をES3（ExtendScript用バージョン）に変換し使えるようにします。

ZIPとしてダウンロードする、リポジトリをフォークするなどしてご利用ください。

詳細は[【解決】モジュール化したExtendScriptを自動で1つにまとめたい！（1）](https://sttk3.com/blog/tips/build-extendscript-1.html)を参照してください。
